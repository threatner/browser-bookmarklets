# 🔎 SEO Meta Inspector Bookmarklet

Review SEO-relevant metadata, social tags, heading structure, and structured data from a single panel.

## What it does

- Shows title, description, canonical URL, and robots directives
- Reports title and description lengths
- Lists key Open Graph tags
- Lists key Twitter card tags
- Summarizes heading counts from `h1` to `h6`
- Counts JSON-LD structured data blocks
- Highlights obvious SEO issues like missing metadata

## How to install

1. Create a new bookmark in your browser
2. Open the source file below
3. Copy the full bookmarklet code and paste it as the bookmark URL

Source: [bookmarklets/seo-meta-inspector.js](../bookmarklets/seo-meta-inspector.js)

## How to use

1. Click the bookmarklet on any page
2. Review the metadata cards and warning list
3. Check whether social tags and headings are present
4. Use the findings to improve on-page SEO basics

## Use cases

- SEO spot checks during content publishing
- Reviewing landing pages before launch
- Comparing metadata across environments
- Auditing competitors or client sites

## Notes

- Missing metadata is not the only SEO problem, but it is a strong first pass
- Title and description length guidance is heuristic, not absolute
